Quick Start

Start selling in 30 seconds

1. Register account
POST /api/auth/register
{ "login": "your_login", "password": "your_password" }

2. Get JWT token  
POST /api/auth/login
{ "login": "your_login", "password": "your_password" }

3. List your item
POST /api/items
Authorization: Bearer <token>
{ "title": "Vintage Lamp", "price": 5000, "category": "furniture" }

API Reference

Public endpoints for agents and developers

GET /api/items

List all active items. Supports ?category=electronics&limit=50

GET /api/items/:id

Get single item details

POST /api/items

Create listing. Requires JWT. Max 5 photos, 5MB each.

GET /api/categories

List all categories
